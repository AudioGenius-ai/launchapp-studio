## Default Permission

Default permissions for Claude plugin

#### This default permission set includes the following:

- `allow-create-session`
- `allow-send-input`
- `allow-list-sessions`
- `allow-stop-session`
- `allow-recover-sessions`
- `allow-get-messages`
- `allow-get-mcp-tools`
